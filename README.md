# Ejercicios de Java

**Proyecto de práctica personal** para aprender y reforzar conceptos fundamentales de Java. Contiene ejercicios de estructuras de datos, condicionales, manejo de cadenas y organización de código en paquetes.

## Contenido

- **DataStructure:** Ejercicios sobre arrays y colecciones básicas.
- **Conditionals Exercises:** Práctica con `if`, `switch` y operadores lógicos.
- **Strings Exercises:** Manipulación y formateo de cadenas de texto.
- **Packages:** Organización del código en paquetes para mejorar la estructura.

Cada carpeta incluye archivos `.java` con implementaciones y ejemplos de uso.
