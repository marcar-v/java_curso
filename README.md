# Ejercicios de Java

Proyecto de práctica personal orientado a aprender y reforzar los conceptos fundamentales del lenguaje Java. El repositorio agrupa ejercicios progresivos centrados en la lógica, la estructura del código y el uso correcto de las bases del lenguaje.

## Contenido

- **DataStructure**  
  Ejercicios sobre arrays y colecciones básicas, manejo de datos y recorridos.

- **Conditionals Exercises**  
  Práctica con estructuras condicionales (`if`, `else`, `switch`) y operadores lógicos.

- **Strings Exercises**  
  Manipulación, comparación y formateo de cadenas de texto.

- **Packages**  
  Organización del código en paquetes para mejorar la modularidad y la estructura del proyecto.

- **LoopsExercises**  
  Uso de bucles (`for`, `while`, `do-while`) aplicados a problemas básicos de iteración.

- **FunctionsExercises**  
  Definición y uso de métodos, paso de parámetros, retorno de valores y reutilización de código.

- **StructureExercises**  
  Ejercicios centrados en la estructura general de programas Java y buenas prácticas básicas.

- **ClassesExercises**  
  Introducción a la programación orientada a objetos: clases, atributos, métodos y encapsulación.

## Notas

Cada carpeta contiene archivos `.java` con implementaciones sencillas y ejemplos de uso, enfocados en consolidar la lógica y la sintaxis antes de avanzar a conceptos más complejos.
